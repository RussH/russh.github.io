## OpenCATS v0.10.0 is available

OpenCATS v0.10.0 is now available from the [OpenCATS GitHub releases page](https://github.com/opencats/OpenCATS/releases/tag/v0.10.0).

Review the release notes and installation documentation before upgrading or installing a new instance:

* [Download the latest release](https://github.com/opencats/OpenCATS/releases/latest)
* [Read the installation documentation](https://documentation.opencats.org)
* [Join GitHub Discussions](https://github.com/opencats/OpenCATS/discussions)
* [Review published security advisories](https://github.com/opencats/OpenCATS/security/advisories)

If you encounter an issue with the release, please report it in the [OpenCATS GitHub issue tracker](https://github.com/opencats/OpenCATS/issues).
